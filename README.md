# Epilepsy Classification using CART Algorithm

## Project Overview
This project explores clinical and physiological features from the **Bangalore Epilepsy dataset** (UCI Repository) to classify outcomes using the CART (Decision Tree) algorithm.

## Key Features
- **Data Preprocessing:** Handles missing values ('?' and 'unknown') and performs Label Encoding and Standard Scaling.
- **Model:** Decision Tree Classifier (CART) with Gini impurity.
- **Interpretability:** Includes visualization of the Decision Tree to understand the classification logic.

## Technical Stack
- Python (Pandas, NumPy, Scikit-learn, Matplotlib)
- Excel (Dataset source)
